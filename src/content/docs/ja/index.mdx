## RTM Wikiへようこそ
RTM Wikiは、RealTrainMod (RTM)のプレイ、アドオン制作などに関する情報をまとめるWikiサイトです。
## RealTrainModとは？
RealTrainMod（以下：RTM）は、Minecraft Java Editionの鉄道Modです。  
日本人のngt5479氏によって開発されました。  
RTMについての詳しい情報は、[RealTrainModの基本情報・魅力](usage/basic)を参照してください。
## ㊗ RealTrainMod 誕生12周年
RTMは2025年7月13日に誕生12周年を迎えました。  
この場を借りてお慶び申し上げます。
